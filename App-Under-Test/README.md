# Aplicación bajo prueba

Calculadora utilizada para realizar las pruebas funcionales.

## Instrucciones

1. Descargar el archivo `calculator.exe`
2. Ejecutar la aplicación
3. Probar las funcionalidades básicas (suma, resta, multiplicación, división)

## Objetivo

Validar el comportamiento de la aplicación y detectar errores en operaciones matemáticas y manejo de inputs.
