# Testing de Calculadora

Este archivo contiene el testing funcional de una calculadora.

## Contenido

- Casos de prueba diseñados
- Resultados de ejecución (pasó / falló)
- Análisis de errores
- Dashboard con visualización de resultados

## Objetivo

Validar el correcto funcionamiento de las operaciones matemáticas y detectar fallas en la aplicación.

## Resultados

Se detectaron múltiples errores en:
- Operaciones de resta
- Ingreso de números
- División
- Estabilidad de la aplicación
